Схемы 2  и 3 абсолютно устойчивы [2]. С использованием выбранных треугольных схем были решены четыре задачи с различными векторами скорости. Задачи решались на временном промежутке t ∈ [0, 20] с шагом по времени, увеличение которого в случае применения схемы 1 приводит к потере устойчивости. Для 2-й и 3-й схем, учитывая их абсолютную устойчивость, шаг по времени равен τ = 20. Решения получены на сетке 32× 32 по пространству при следующих числах Пекле:  1. Pe = 10−1 — доминирующая диффузия; 2. Pe = 10 — средняя диффузия; 3. Pe = 103 — слабая диффузия; 4. Pe = 105 — доминирующая конвекция.  В таблицах 1 - 3 представлены численные результаты решения задачи (1): приведены шаг по времени, время, затраченное на решение задачи на отрезке [0, 20], и относительная погрешность решения в норме пространства C на последнем шаге по времени. Численные расчеты подтвердили полученные теоретические результаты [2] об условной устойчивости схемы (2) с B = BC +2τKH (схема 1) и абсолютной устойчивости схемы (2) с B = (1 + ατ/2)E+τKH , α = ‖M‖ (схема 2) и B = (1 + αiτ/2)E+τKH , αi =  N∑ j=1  |mij | (схема 3). Схема 1 при доминирующей диффузии неустойчива, возрас тание влияния конвекции ведет к увеличению шага по времени, при котором схема сохраняет устойчивость. Схемы 2 и 3 абсолютно устойчивы независимо от того, какой из физических процессов доминирует, и решают задачу за гораздо меньшее время, чем схема 1.  Как и ожидалось, погрешность решения задачи с гладким точным решением гораздо меньше, чем в случае решения типа погранслоя.  При увеличении числа Pe точность решения задачи с гладким решением и с решением типа погранслоя ухудшается для всех трех схем.
