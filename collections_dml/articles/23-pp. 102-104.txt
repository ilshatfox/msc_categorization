В Lp рассмотрим классы функций: L̃p = {f : f̃ ∈ Lp}, L{λn}p = {f : f{λn} ∈ Lp}; Eϕp = {f : En(f)p = O(ϕ(n−1))}; Hk,ϕp = {f : ωk(f, δ)p = O(ϕ(δ)},  где, как и всюду ниже, мажоранта ϕ(δ) непрерывна на [0, 2π], ϕ(0) = = 0, 0 < ϕ(δ1) < ϕ(δ2) при 0 < δ1 ≤ δ2 (ϕ ∈ Φ, см. [2], [3]). Справедливы следующие утверждения.  Утверждение 1. Пусть ϕ ∈ Φ; p = 1,∞. Тогда Eϕp ⊂ L̃p ⇔ Eϕp ⊂ L{ln(n+2)}p .  Утверждение 2. Пусть ϕ ∈ Φ, δ−kϕ(δ) не возрастает; p = 1,∞. Тогда  Hk,ϕp ⊂ L̃p ⇔ Hk,ϕp ⊂ L{ln(n+2)}p .  Следствие. Пусть ϕ ∈ Φ (δ−kϕ(δ) не возрастает); p = 1,∞. Тогда для того, чтобы для всякой функции f ∈ Lp из En(f)p = O(ϕ(n−1)) (ωk(f, δ)p = O(ϕ(δ))) следовало f̃ ∈ Lp необходимо, чтобы an ln(n + 2) → → 0, bn ln(n + 2) → 0 (n →∞), где an, bn — коэффициенты Фурье функции f ∈ Lp.
