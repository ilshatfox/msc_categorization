) ХАРАКТЕРИЗАЦИЯ СЛЕДА  СТЕПЕННЫМИ НЕРАВЕНСТВАМИ  Всюду ниже через Mn обозначаем линейное пространство всех n×nматриц над полем комплексных чисел, через Mhn — подмножество эрми 45    товых матриц, через M+n — подмножество M h n положительно полуопреде ленных матриц. Для X,Y ∈ Mhn запись X ≤ Y означает, что Y −X ∈ M+n . Для матрицы X ∈ Mhn и вещественной функции f , область определения которой содержит спектр X, по конечномерной версии спектральной теоремы определяется значение f(X) ∈ Mhn . В частности, для X ∈ M+n и любого p > 1 корректно определяется матрица Xp. Для X, Y ∈ Mhn абсолютное значение |XY | определяется как (Y X2Y )1/2. Через M ′+n обозначаем множество линейных функционалов на Mn, принимающих неотрицательные значения на M+n .  Теорема. Для функционала ϕ ∈ M ′+n выполнение каждого из следующих условий влечет, что ϕ является скалярным кратным следа:  (i) для некоторых чисел p, q > 1, 1 p + 1  q = 1, p 6= 2, для любых матриц  X, Y ∈ M+n выполняется “неравенство Гельдера”  |ϕ(XY )| ≤ ϕ(Xp)1/pϕ(Y q)1/q;  (ii) для некоторых чисел p, q > 1, 1 p  + 1 q  = 1, для любых матриц X, Y ∈ M+n выполняется “неравенство Юнга”  ϕ(|XY |) ≤ 1 p ϕ(Xp) +  1  q ϕ(Y q);  (iii) для некоторого p > 1 для любых таких матриц X,Y ∈ M+n , что X ≤ Y , выполняется “неравенство монотонности” ϕ(Xp) ≤ ϕ(Y p).  Как известно, для следа вышеупомянутые неравенства имеют место и, таким образом, каждое из условий (i), (ii), (iii) характеризует след.  Следствие. Функционал ϕ ∈ M ′+n является скалярным кратным следа тогда и только тогда, когда ϕ(eX) ≤ ϕ(eY ) для любых таких X, Y ∈ Mhn , что X ≤ Y .  Заметим еще, что приведенные результаты допускают обобщения на случай функционалов и весов на C*- и W*-алгебрах.
