.  ДВОЙСТВЕННАЯ РЕГУЛЯРИЗАЦИЯ В ОБРАТНОЙ ЗАДАЧЕ ФИНАЛЬНОГО НАБЛЮДЕНИЯ ДЛЯ УРАВНЕНИЯ  ТЕПЛОПРОВОДНОСТИ  Ф. А. Кутерин  Нижегородский государственный университет, xredor@yandex.ru  Доклад посвящен обсуждению алгоритма двойственной регуляризации, а именно, регуляризации неустойчивого к ошибкам исходных данных классического двойственного алгоритма Удзавы [1] для решения обратной задачи теплопроводности. Алгоритм заключается в непосредственном решении на основе градиентного метода двойственной к исходной обратной задачи и предназначен для восстановления распределенной правой части  83    уравнения теплопроводности. Обсуждается итеративная регуляризация предлагаемого алгоритма. Приводятся результаты работы компьютерной программы, реализующей алгоритм итеративной регуляризации.  Данный алгоритм предназначен для восстановления правой части уравнения.  Пусть D ≡ {u ∈ L∞(QT ) : u(x, t) ∈ U п. в. на QT }, U — замкнутое выпуклое множество в Rm, QT ≡ (a, b) × (0, T ). Рассматривается обратная задача финального наблюдения в случае первой краевой задачи для уравнения теплопроводности  zt − zxx + u(x, t) = 0, z(x, 0) = z0(x), x ∈ (a, b), (1)  z(a) = z(b) = 0, заключающаяся в приближенном нахождении нормального, т. е. минимального по норме, коэффициента (распределенной правой части) u ∈ (D) по приближенно известному в финальный момент времени наблюдению q = z(·, T ) ∈ L2(a, b).  Эту задачу можно переформулировать в виде эквивалентной задачи оптимального управления:  I0(u) → inf, I1(u) = q, u ∈ D, q ∈ H = L2(Ω), (Pq)  где I0(u) = ||u||22,QT , I1 = z[u](·, T ). С учетом приближенного задания исходных данных мы имеем вместо задачи (Pq) семейство зависящих от ошибки их задания δ задач  I0(u) → inf, I1(u) = hδ + q, u ∈ D, q ∈ H = L2(Ω). Важно отметить, что для классического алгоритма Удзавы необходимо существование седловой точки функционала Лагранжа соответствующей задачи оптимального управления. Предлагаемый алгоритм работает как в случае существования седловой точки, так и в случае её отсутствия. Кроме того, двойственный алгоритм в отличие от классического алгоритма Удзавы устойчив к ошибкам исходных данных.  Рассматриваются также примеры решения конкретной обратной задачи описанным методом.
