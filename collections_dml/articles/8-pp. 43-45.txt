) АЛГЕБРАИЧЕСКИ ПОРОЖДАЕТСЯ ПОЛУОРТОГОНАЛЬНЫМИ ПРОЕКТОРАМИ  ПустьH — гильбертово пространство над полем Λ(= R или C). Через B(H) обозначим ∗-алгебру всех линейных ограниченных операторов в H. Оператор T ∈ B(H) называется — идемпотентом, если T 2 = T ; — проектором, если T 2 = T = T ∗; — полуортогональным проектором [1], если  T ∗T = 1 2 (T + T ∗). (1)  Обозначим через I, P, S множества всех идемпотентов, проекторов и полуортогональных проекторов B(H), соответственно. Тогда I∩S=P.  В [2] доказано, что каждый оператор T ∈ B(H) представляется в виде конечной суммы T =  ∑ Tk, где каждое Tk есть  произведение не более, чем двух проекторов при dim H = ∞ и не более, чем трех проекторов при 2 ≤ dim H < ∞.  43    Теорема 1. Пусть A — C∗-подалгебра в B(H) с единицей I. Каждый элемент T ∈A представляется в виде конечной суммы T =  ∑ Tk с Tk ∈ {−I} ∪ ( A∩S).  Из (1) ясно, что оператор −I не представляется в виде конечной суммы элементов из A∩S.  Каждый оператор из B(H), dim H = ∞, представляется в виде конечной суммы не более, чем пяти идемпотентов [3]. Для конечномерного пространства имеет место лишь более слабый факт [4]: каждый линейный оператор представляется в виде конечной суммы попарных произведений идемпотентов (в каждом слагаемом один из сомножителей можно выбрать эрмитовым).  Теорема 2. Каждый оператор T ∈ B(H), dim H ≥ 2, представляется в виде конечной суммы T =  ∑ Tk, где каждое Tk  есть произведение не более, чем двух полуортогональных проекторов.  Этот результат неулучшаем: при Λ = R и dim H= 2 канонический след tr(PQ) = tr(QP ) ≥ 0 для всех P ∈P, Q ∈S. Отметим также, что уже в двумерном случае доказанное в [5] равенство ‖P + Q‖ = 1 + ‖PQ‖ (P,Q ∈P) не распространяется на полуортогональные проектора.
