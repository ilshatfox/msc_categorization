-МОНОТОННЫЕ БАНАХОВЫ ПАРЫ  В докладе будет обсуждаться понятие K-монотонности, играющее важную роль в теории интерполяции операторов. Для банаховой пары (X0, X1), x ∈ X0 + X1 и t > 0 определим Kфункционал Петре:  K(t, x; X0, X1) = inf{||x0||X0 + t||x1||X1 : x = x0 + x1, xi ∈ Xi}.  Пару (X0, X1) называютK-монотонной (парой Кальдерона-Митягина), если для некоторого C > 0 и всех x, y ∈ X0 + X1 из неравенства  K(t, y; X0, X1) ≤ K(t, x; X0, X1) (t > 0) вытекает существование линейного оператора T : X0 + X1 → X0 + X1, такого, что maxi=0,1 ‖T‖Xi→Xi ≤ C и y = Tx. Если последнее выполнено при C = 1, то (X0, X1) называется точной K-монотонной парой. Если пара K-монотонна, то все пространства, интерполяционные относительно нее, описываются как пространства вещественного K-метода.  В середине 60-х годов А. П. Кальдерон [1] и независимо от него Б. С. Митягин [2] получили первый важный пример точной K-монотонной пары — (L1, L∞). Вскоре выяснилось, что этим свойством, более того, обладает любая пара ”весовых” Lp-пространств (Lp0(w0), Lp1(w1)) (1 ≤ p0, p1 ≤ ∞) [3]–[5].  21    Наряду с этим в работе [3] был приведен пример пары пространств размерности 3, не имеющей свойства точнойK-монотонности. Приведем основные результаты доклада, содержащие необходимые и достаточные условия точной K-монотонности пар вида (λn(v), ln∞) и (Λ(ϕ), L∞)).  Теорема 1. Банахова пара (λn(v), ln∞) является точной Kмонотонной тогда и только тогда, когда vk = v1qk−1 (k = 1, 2, .., n), где v1 > 0, а q ∈ [0, 1].  Теорема 2. Пара (Λ(ϕ), L∞) пространств на полуоси (0,∞) обладает свойством точной K-монотонности тогда и только тогда, когда ϕ(t) = af(t) (a > 0), где либо f(t) = 1, либо f(t) = t, либо f(t) = 1− qt (0 < q < 1).
