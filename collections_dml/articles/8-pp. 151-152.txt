 ЦЕНТРАЛЬНОЙ ПРЕДЕЛЬНОЙ ТЕОРЕМЕ ДЛЯ МУЛЬТИПЛИКАТИВНОЙ ОРТОГОНАЛЬНОЙ  СИСТЕМЫ ФУНКЦИЙ  Через χn(x), n = 0, 1, 2, · · ·; x ∈ [0, 1], обозначим периодическую мультипликативную ортонормированную систему функций (сокращенно — ПМОНС), определенную с помощью последовательности простых чисел {pk} (подробное определение см., например, в [1]).  Пусть также {nk} — некоторая лакунарная последовательность номеров, подчиненная (начиная с некоторого k) условию:  nk+1 nk  ≥ 1 + ω(k) при k = 1, 2, ...,  где ω(k) монотонно убывает (но не быстрее некоторой степени kα, 0 < α < 1).  Определим величины: f(0) = 0, f(k + 1) = max{i : ni < mk}, k = 0, 1, 2, . . .; Bk =  √ f(k + 1), p̃k = max {pi : 1 ≤ i ≤ k + 1}.  В [2] исследовался вопрос о подчиненности последовательности {χnk(x)} (трактуемой как последовательность двумерных случайных векторов) центральной предельной теореме. Были получены (теорема A) достаточные условия выполнения ЦПТ в случае, когда числа pk, структурирующие систему, ограничены в совокупности.  151    Следующий результат распространяет теорему A на случай, когда limk→∞pk = ∞.
