Ключевые слова: дуальная гипотеза Смейла, критические точки, критические значения полиномов.  В 1981 году Смейл [1] выдвинул следующую гипотезу. Пусть f - полином, deg f ≥ 2, такой что f (0) = 0 и f ′(0) = 1. Обозначим  S( f ) = min {∣∣∣∣ f (ζ)ζ  ∣∣∣∣ : f ′(ζ) = 0 }  ,  Kn = sup{S( f ) : deg f = n, f (0) = 0, f ′(0) = 1}. Гипотеза утверждает, что Kn = 1−1/n. Доказаны следующие неравенства:  1− 1 n ≤ Kn < 4.  Существует большое количество оценок для различных частных случаев. Некоторые из них можно найти в статье [2].
