Лапшина  Самарский государственный университет,  marijalapshina@rambler.ru  Набор векторов {f}Mi=1 ⊂ RN называется фреймом Парсеваля, если для каждого f ∈ RN справедливо равенство  M∑  i=1  | < f, fi > |2 = ‖f‖2.  Если ‖f1‖ = ‖f2‖ = . . . = ‖fM‖, то ‖fi‖2 = N/M. Фреймы Парсеваля с одинаковыми нормами представляют большой интерес  для прикладных исследований. Известно [1], что для каждого  M > N существует фрейм Парсеваля с одинаковыми нормами,  однако построение таких фреймов требует сложных алгорит мов.  В работе приведены простые способы построения фреймов  Парсеваля с одинаковыми нормами для следующих пар: M =  3, N = 2; M = 4, N = 2; M = 5, N = 2, N = 3, N = 4;  M = 6, N = 3.
