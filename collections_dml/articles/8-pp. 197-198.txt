)  ПРИБЛИЖЕНИЯ ПОЛИНОМАМИ В ХАУСДОРФОВОЙ МЕТРИКЕ  Пусть f(x) — ограниченная 2π-периодическая, вообще говоря, многозначная функция с дополненным графиком, δ > 0,  s(f, δ; x) = min{sup{f(t) : x−δ < t ≤ x}, sup{f(t) : x ≤ t < x+δ}}, i(f, δ;x) = max{inf{f(t) : x− δ < t ≤ x}, inf{f(t) : x ≤ t < x+ δ}},  µ(f ; δ) = inf x {s(f, δ;x)− i(f, δ; x)}/2.  Если в определении s(f, δ; x) поменять min на max, а в определении i(f, δ;x) — max на min, то получатся величины S(f, δ; x) и I(f, δ;x) соответственно, которые посредством равенства  ω(f ; δ) = sup x {S(f, δ/2;x)− I(f, δ/2; x)}  определяют равномерный модуль непрерывности ω(f ; δ) функции f .  197    Введём теперь следующую характеристику функции f :  R(f ; δ) = ω(f ; δ) + δ µ(f ; δ) + δ  .  Положим также R∗(f ; δ) = sup{R(f ; t) : t ≥ δ}. В терминах R(f ; δ) и R∗(f ; δ) можно указать новые, и при этом асимптотически точные, оценки наименьших уклонений HEn(f) функции f от тригонометрических полиномов порядка не выше n в метрике Хаусдорфа.  Теорема. При всех достаточно больших n (n > n(f)) имеем  HEn(f) ≤ inf {  δ : 1 δ  ( ln R(f ; δ) +  5 2  ln ln (eR(f ; δ)) + 11 )  < n }  .  На классе ограниченных функций постоянная 5/2 перед ln ln (eR(f ; δ)) не может быть заменена на величину, меньшую −1/2.  Следствие 1.
