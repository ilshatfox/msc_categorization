.  НЕСИММЕТРИЧЕСКИЕ МЕТРИЧЕСКИЕ СВЯЗНОСТИ И ИХ КЛАССИФИКАЦИЯ  И. А. Гордеева  Владимирский государственный педагогический  университет, gordeeva@vgpu.vladimir.ru  Рассмотрим n-мерное (псевдо)риманово многообразие M с  метрическим тензором g и связностью Леви-Чивиты ∇. Заданная на M линейная связность ∇ с ненулевым кручением называется несимметрической метрической связностью, если  ∇g = 0. Обозначим через S тензор кручения связности ∇. Тогда тензор S, такой, что S(Z,X, Y ) = g(Z, S(X,Y )) для любых  X,Y,Z ∈ TM , в каждой точке x ∈ M удовлетворяет условию Sx ∈ T ∗xM  ⊗ Λ2(TxM).  Рассмотрим два оператора  Λ3 : T ∗xM ⊗  Λ2(T ∗xM) → Λ3(T ∗xM)  и  (tr12W )(X) = ∑  k=1,...,n  W (ek, ek, X)  для произвольного Wx ∈ T ∗xM ⊗  Λ2(T ∗xM) и ортонормирован ного базиса {e1, . . . , en} пространства TxM . Тогда справедливо разложение  T ∗xM ⊗  Λ2(TxM) = gx ∧RT ∗xM ⊕Λ3(T ∗xM) ⊕ ker λ3 ∩ ker(tr12),  которое является неприводимым относительно действия орто гональной группы O(gx). В соответствии с этим разложением  62    инвариантным образом выделим семь классов несимметриче ских метрических связностей. При этом к одному классу бу дем относить все связности ∇, для которых тензор Sx для всех x ∈M принадлежит одному и тому же инвариантному подпространству из T ∗xM ⊗ Λ2(T ∗xM).  Будем полагать ∇ ∈ C1, если Sx ∈ gx ∧ RT ∗xM для всех x ∈M . Тогда  S(Z,X, Y ) = (n− 1)−1[g(Z,X)(tr12S)(Y ) − g(Z, Y )(tr12S)(X)]  для любых X,Y,Z ∈ TM . Этот класс состоит из полусимметрических метрических связностей, которые были введены в  рассмотрение К. Яно (см. [1]) и продолжают вызывать интерес  исследователей вплоть до последнего времени (см., например,  [2,3]).  Будем полагать ∇ ∈ C2, если Sx ∈ Λ3(TxM) для всех x ∈M . Этот класс связностей характеризуется тем, что средняя связ ность взаимной пары связностей (см. [4], с. 129) совпадает  со связностью Леви-Чивиты, т. е. 2−1(∇XY + ∇YX) = ∇XY для любых X,Y ∈ TM . Так же нетрудно доказать, что связности ∇ ∈ C2 имеют общие геодезические со связностью ЛевиЧивиты ∇.  Такие связности рассматривал К. Яно (см. [5], с. 78–92) с  целью расширения области применения «техники Бохнера», и  им же было доказано, что связности ∇ ∈ C2 естественным образом возникают в пространстве полупростой группы Ли (см.  там же).  Следующие классы несимметрических метрических связно стей не были исследованы:  1) ∇ ∈ C3, что означает Sx ∈ ker Λ3 ∩ ker(tr12S) для всех x ∈M ; тогда  S(Z,X, Y ) + S(X,Y,Z) + S(Y,Z,X) = 0  63    и  (tr12S)(X) = ∑  k=1,...,n  S(ek, ek, X) = 0;  2) ∇ ∈ C1 ⊕ C3, что означает Sx ∈ kerΛ3 для всех x ∈ M ; тогда  S(Z,X, Y ) + S(X,Y,Z) + S(Y,Z,X) = 0;  3) ∇ ∈ C1 ⊕ C2, что означает Sx ∈ gx ∧ RT ∗xM ⊕ Λ3(T ∗xM) для всех x ∈M ; тогда  S(Z,X, Y ) + S(X,Z, Y ) =  = (n− 1)−1(2g(Z,X)(tr12S)(Y )−  − g(X,Y )(tr12S)(Z) − g(Z, Y )(tr12S)(X));  4) ∇ ∈ C2 ⊕C3, что означает Sx ∈ ker(tr12) для всех x ∈M ; тогда  (tr12S)(X) = ∑  k=1,...,n  S(ek, ek, X) = 0.
