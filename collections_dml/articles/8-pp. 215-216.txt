)  ЧИСЛЕННО-АНАЛИТИЧЕСКОЕ РЕШЕНИЕ БИГАРМОНИЧЕСКОГО УРАВНЕНИЯ  Рассматривается краевая задача для бигармонической функции в многосвязной области.Известно, что бигармоническая функция может быть выражена через две аналитические функции, которые могут быть найдены известными методами теории аналитических функций. В частности, в случае двусвязных областей для решения задачи можно использовать конформное отображение на кольцо параметрической плоскости и отыскивать аналитические функции в виде ряда Тейлора.  Предлагается два способа определения коэффициентов ряда: 1) сравнение коэффициентов при одинаковых тригонометрических функциях и 2) удовлетворение граничным условиям в отдельных точках (метод коллокации). Первый способ позволяет в некоторых случаях получить аналитическое решение, второй — разработать численный алгоритм расчета достаточно сложных задач. Численные примеры подтверждают удовлетворительное совпадение обоих способов.  Для областей с произвольными, но непересекающимися границами разработан численный алгоритм, основанный на применении метода граничных элементов к интегральному уравнению для бигармонической функции [1]. Задача сводится к решению системы линейных уравнений. Метод может быть применен к численному решению бигармонического уравнения в многосвязной области.
