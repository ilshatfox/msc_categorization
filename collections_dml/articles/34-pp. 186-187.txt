Сайфутдинов  Казанский государственный университет, ilnursoft@mail.ru  Рассматривается задача об излучении из полубесконечного  цилиндрического волновода в волновод меньшего радиуса, ось  симметрии которого смещена относительно оси первого вол новода. Цилиндрические волноводы с идеально проводящими  стенками заполнены однородной изотропной средой. Свойства  сред в волноводах в общем случае различны. Пусть из волно вода с бо́льшим диаметром набегает на стык собственная волна  TM0,1. Необходимо найти электромагнитное поле в каждой ча сти волноводной структуры. Предполагается выполнение гипо тез классической электродинамики для электромагнитных по лей и независимость поля от угла α. В связи с тем, что поле  не зависит от угла α, будем рассматривать процесс в плоско сти (r, z). Компоненты падающей электромагнитной волны из вестны, а параметры рассеянного поля можно выразить через  Ez = u, где данная фукция удовлетворяет волновому уравне нию 1  r  ∂  ∂r  ( r ·  ∂u  ∂r  ) +  1  r2 ∂2u  ∂α2 + (k2 + γ2)u = 0 (1)  188    с граничными условиями на стенках волновода  E0,1z |r=a= 0, z < 0, ∀α, E0,1z |r′=b= 0, z > 0, ∀α′,  и в плоскости соединения двух волноводов касательные состав ляющие полного электромагнитного поля непрерывны. Гра ничные условия исходной задачи будут выполнены, если соб ственные фунции представить в каждой из областей системы  в виде разложения по собственным волнам, уходящим от плос кости z на бесконечность. При решении задачи использовался  метод сшивания на границе раздела частичных областей, по лучено парное функциональное уравнение относительно коэф фициентов разложения искомых полей по собственным волнам  волноводов. Для регуляризации функционального уравнения  использован метод интегральных тождеств. Исходная задача  сведена к бесконечной системе линейных алгебраических ура нений, численное решение которой найдено методом усечения.
