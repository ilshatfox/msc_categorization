Предположим, что след τN — ограничение τM на N — также полуконечен. Обозначим через L1(τM ) и L1(τN ) пространства операторов, интегрируемых относительно τM и τN соответственно, а через K(τM ) и K(τN ) — соответствующие пространства вполне измеримых операторов. Совокупность множеств:  K(², δ, p) = {x ∈ K(τM ) |∃q ∈ Mpr (q ≤ p, ‖qxq‖∞≤ ², τ(p−q) ≤ δ)}, где ², δ > 0, p ∈ Mpr, τ(p) < ∞, образует базис окрестностей нуля отделимой топологии в K(τM ), согласованной с линейной структурой. Эту топологию назовем топологией локальной сходимости по мере.  Теорема 1 [3]. Пусть X — такое подпространство L1(τM ), что единичный шар X замкнут в топологии локальной сходимости по мере. Тогда факторпространство L1(τM )/X слабо секвенциально полно.  Доказательство этой теоремы в основных чертах переносится из работы Годфруа [1], где рассматриваются коммутативные L1пространства, только вместо использованной там теоремы Бухвалова–Лозановского используется ее некоммутативная версия из [2].  Теорема 2.Факторпространство L1(τM )/L1(τN ) слабо секвенциально полно.  Теорема 2 немедленно следует из теоремы 1, если заметить, что единичный шар пространства L1(τN ) замкнут в K(τM ) в топологии локальной сходимости по мере. Для случая конечного следа этот факт следует из полноты пространства K(τN ) в топологии сходимости по мере.
