 ОДНОРОДНЫХ ЛОГИКАХ ГРИЧИ С ЕДИНСТВЕННЫМ СОСТОЯНИЕМ  Пусть n,m — натуральные числа. Квантовая логика (ортомодулярное частично упорядоченное множество, OMP) L называется (n,m)-однородной [1], если каждый ее атом содержится в  213    n блоках (блок— максимальное по включению семейство попарно ортогональных атомов), а каждый блок в L содержит ровно m атомов логики L.  Пусть L — конечная (n,m)-однородная логика. Обозначим через A — множество всех атомов L, через B — множество всех блоков L и через S = S(L) — множество всех состояний на L. Нетрудно видеть, что в такой логике n card A = m card B и существует состояние, равное 1/m на любом атоме. Нас интересуют однородные логики L для которых S(L) одноэлементно. Исследования ортомодулярных решеток с заданным свойствами для S было проведено в ряде работ М. Навары, П. Птака и В. Рогалевича. Был известен пример OMP с 22 атомами с единственным состоянием (Миллер, Гричи). Нами построен пример (3, 3)-однородной логики L17 c 17 атомами, допускающей единственное состояние.  Обозначим через ai, i = 0, 1, ..., 16 атомы L17. Опишем множество блоков B этой логики: семь блоков, задающие петлю [2] порядка 7 имеют вид {a2i, a2i+1, a2i+2} (индексы по модулю 14); для остальных 10 блоков мы выпишем только индексы у атомов  { 0, 8, 14 }; { 1, 5, 11 }; { 1, 7, 16 }; { 2, 9, 15 }; { 3, 7, 13 }; { 3, 11, 14 }; { 4, 10, 16 }; { 5, 9, 13 }; { 6, 12, 15 }; {14, 15, 16}. Теорема. cardS(L17) = 1.  Замечание. 1. (3, 3)-однородные логики возникают также, когда мы исследуем релятивные OMP на конечных множествах. Например, на 8-элементном множестве релятивная логика L имеет в качестве горизонтальных слагаемых (3, 3)-однородные логики с 28 атомами. Для этой логики также cardS(L) = 1.  2. Недавно М. Терентьев (Чебоксары)  ЧИСЛЕННО-АНАЛИТИЧЕСКОЕ РЕШЕНИЕ БИГАРМОНИЧЕСКОГО УРАВНЕНИЯ  Рассматривается краевая задача для бигармонической функции в многосвязной области.Известно, что бигармоническая функция может быть выражена через две аналитические функции, которые могут быть найдены известными методами теории аналитических функций. В частности, в случае двусвязных областей для решения задачи можно использовать конформное отображение на кольцо параметрической плоскости и отыскивать аналитические функции в виде ряда Тейлора.  Предлагается два способа определения коэффициентов ряда: 1) сравнение коэффициентов при одинаковых тригонометрических функциях и 2) удовлетворение граничным условиям в отдельных точках (метод коллокации). Первый способ позволяет в некоторых случаях получить аналитическое решение, второй — разработать численный алгоритм расчета достаточно сложных задач. Численные примеры подтверждают удовлетворительное совпадение обоих способов.  Для областей с произвольными, но непересекающимися границами разработан численный алгоритм, основанный на применении метода граничных элементов к интегральному уравнению для бигармонической функции [1]. Задача сводится к решению системы линейных уравнений. Метод может быть применен к численному решению бигармонического уравнения в многосвязной области.  В качестве приложения рассмотрено движение тел различной конфигурации в ограниченной предельно вязкой жидкости [2].  215
