)  ВОПРОСЫ ЕДИНСТВЕННОСТИ ДЛЯ НЕКОТОРОГО КЛАССА РЯДОВ ХААРА  Мы рассматриваем некоторые вопросы единственности для  рядов Хаара ∞∑  n=1 anχn(x). Используется следующая терминоло гия. Множество A ∈ [0, 1] называется М-множеством для рядов Хаара с некоторым условием (a), если существует ряд Хаара с условием (a), не все коэффициенты которого нулевые, удовлетворяющий условию (a) и сходящийся к нулю вне A. Если множество A не является М-множеством для рядов Хаара с условием (a), то множество A назовем U-множеством для рядов Хаара с условием (a). В качестве условия (a) рассмотрим условие  SN (x) = o(N 1−α), (1)  где SN (x) = N∑  n=1 anχn(x), o — ”свое” для каждого x, а 0 ≤ α < 1.  ПустьB — непустое совершенное подмножество отрезка [0, 1]. Введем ряд числовых характеристик множества B. Построим функцию βB(∆), определенную на всех двоичных интервалах ∆. Условимся, что βB(∆) = 0 тогда и только тогда, когда внутри ∆ нет точек из B. Пусть βB([0, 1]) = 1. Далее строим функцию βB по индукции по рангу двоичного интервала ∆ (см. [1]). Пусть βB определена на двоичном интервале ∆ ранга n, определим βB на неперекрывающихся двоичных интервалах ∆i (i = 1, 2), составляющих в объединении∆. А именно, пусть k — число интервалов ∆i, внутри которых есть точки из B, тогда βB(∆i) = 1/kβB(∆), если внутри ∆ есть точки из B и βB(∆i) = 0 в противном случае.  Определение. Назовем индексом точки x ∈ B относительно множества B число (или, возможно, два числа) indB x = sup{α ≥ 0, βB(∆n)= o(∆n)α}, где ∆n — последовательность вложенных двоичных интервалов, внутри каждого из которых есть точки из B, причем ранг ∆n равен n и x ∈ ∆n. При этом два индекса имеют те точки x, для которых существуют две подобные последовательности. Назовем верхним ин 182    дексом множества B число indB = sup x∈B  indB x, а нижним ин дексом множества B — число indB = inf x∈B  indB x. При этом,  0 ≤ indB ≤ indB ≤ 1. Сформулируем основные результаты.  Теорема 1. Если A — М-множество для рядов Хаара с условием (1), то A содержит совершенное подмножество B с indB > α, то A — М-множество для рядов Хаара с условием (1).  Теорема 2. Если A не содержит совершенных подмножеств B с indB ≥ α, то A — U-множество для рядов Хаара с условием (1). В частности, если A — совершенное множество с indA < α, то A — U-множество для рядов Хаара с условием (1).  Теорема 3. Для любых 0 ≤ α ≤ β ≤ 1 существует совершенное множество B с indB = β и indB = α.  Теоремы 2 и 3 показывают, что существуют совершенные U множества для рядов Хаара с условием (1), если 0 < α < 1. Тем самым, обобщается теорема 2 из работы [2], где совершенное Uмножество строится при более сильном условии, а именно, при условии ограниченности общего члена ряда Хаара (в этом случае SN (x) = O(log2 N)).
