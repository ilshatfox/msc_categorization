)  ПРИМЕНЕНИЕ ИНТЕГРАЛЬНЫХ УРАВНЕНИЙ К ПОСТРОЕНИЮ ФУНКЦИЙ РИМАНА  В [1], [2] с помощью интегральных уравнений построены некоторые функции Римана. В настоящем сообщении эти результаты обобщаются на случаи менее ограничительных предположений относительно коэффициентов исходных дифференциальных уравнений. Например, в [1] указанный вопрос решен для уравнения uxy + aux + buy + cu = 0, когда a = a1(y) + λx, b = b1(x) + λy, λ = const. Теперь же показано, что для a = a(x, y), b = b(x, y) достаточно тождества ax ≡ by. Аналогичное обобщение имеет место для уравнения с тремя независимыми переменными.  В плоском случае рассуждения удается распространить на эллиптическое уравнение uxx + uyy + aux + buy + cu = 0 с аналитическими вещественными коэффициентами. Здесь тождество ax ≡ by должно быть заменено на ay ≡ bx. Если при этом функция ln(ax + by + a2 + b2 − 2c) является гармонической, то можно построить аналитическую функцию ϕ(z), z = x + iy, с помощью которой функция Римана записывается в виде  J0  (∣∣∣ ∫ z  t  ϕ(ξ)dξ ∣∣∣ )  exp (∫ z̄  t̄  A(t, η)dη + ∫ z  t  B(ξ, z̄)dξ ) .  Здесь A(z, z̄) = 1 4  [ a (z + z̄  2 , z − z̄  2i  ) + ib  (z + z̄ 2  , z − z̄  2i  )] , B(z, z̄) от личается от A(z, z̄) лишь знаком перед вторым слагаемым, J0(ω) — функция Бесселя.
